MATERI 1: Jajar Genjang

1.1 Sifat Jajar Genjang

1) Pada setiap Jajargenjang sisi-sisi yang berhadapan sama panjang dan sejajar. Pada gambar berarti:

AB = CD AB // CD

BC = DA BC // DA

2) Pada setiap jajargenjang sudut-sudut yang berhadapan sama besar. Pada gambar berarti:

< ABC = < CDA

< BAD = < DCB

3) Pada setiap Jajargenjang jumlah besar sudut-sudut yang berdekatan adalah 180˚. Pada gambar berarti :

< A + < D = 180˚

< A + < B = 180˚

< B + < C = 180˚

< C + < D = 180˚

4) Kedua diagonal setiap Jajargenjang saling membagi dua sama panjang. Pada gambar berarti :

OA = OC

OB = OD

1.2 Rumus

• Luas jajargenjang sama dengan hasil kali alas dan tinggi. L = a x t

• Keliling Jajargenjang sama dengan dua kali jumlah panjang sisi yang saling berdekatan. K = 2(a+b)

1.3 Contoh

1. Diketahui jajar genjang PQRS. Bila luas jajar genjang tersebut adalah 144 cm2 dan panjang PQ adalah 18 cm. Tinggi jajar genjang tersebut adalah?

JAWAB :

L = a x t

144 = 18 x t

t = 144/8

t = 8 cm
